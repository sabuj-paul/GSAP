About GSAP

The Gold Standard for Web Animation

GreenSock Animation Platform (GSAP) is the most robust, high-performance JavaScript animation library available. Used by major companies like Google, Apple, Nike, and NASA, GSAP enables developers to create smooth, complex animations with minimal effort.

  • ✓ Animate any property of any DOM element
  • ✓ Stagger animations across multiple targets
  • ✓ Integrate with React, Vue, and Angular
  • ✓ Free for most use cases
